# Challenger Project #1

# Auction Simulator

## Module

Module 7 - JavaScript Functions (Challenger Project)

---

# Purpose

This project is designed to test the practical understanding of Module 7 concepts.

Unlike the Shopping Cart Training Project, the Auction Simulator introduces a completely different problem domain to ensure concepts are understood rather than memorized.

The objective is to build an event-driven console application where auctions progress over time and users interact before the timer expires.

# Project Objective

Create a console-based auction system where users can:

- Create auctions
- View active auctions
- Place bids
- Track the highest bidder
- Wait while the auction timer runs
- Automatically close auctions when time expires

---
